Tokens
======
Comments
    LineComment              - // continues to end of line
                             - //.*\n
    BlockComment             - { all the text within the curleys is comment }
                             - {[^}]*}
Identifier                   - Up to 256 characters case sensitive.  main and print are primitive identifiers
                             - [a-zA-Z]+
Literal
    IntegerLiteral           - -2^32 to 2^32-1
                             - (-)?[0-9]+
    BooleanLiteral           - true or false
                             - (true|false)
Keyword
    IntegerType              - integer
    BooleanType              - boolean
    IfKeyword                - if
    ThenKeyword              - then
    ElseKeyword              - else
    NotOperator              - not
    OrOperator               - or
    AndOperator              - and
    MainIdentifier           - main
    PrintIdentifier          - print

Operator or punctuator
    PlusOperator             - +
    MinusOperator            - -
    MultiplicationOperator   - *
    DivisionOperator         - \
    LessThanOperator         - <
    EqualityOperator         - =
    OpenBracket              - (
    CloseBracket             - )
    Comma                    - ,
    Colon                    - :


Notes
=====

minus (-)  is used twice
