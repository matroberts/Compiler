KleinPrograms
=============

A collection of sample klein programs.

1) kleins.ps1
a script to print the tokens from a klein program
usage: powershell kleins.ps1 filename
eg: powershell C:\github\Compiler\KleinPrograms\kleins.ps1 C:\github\Compiler\KleinPrograms\Programs\scanner\testcases\reserved-words-and-symbols.kln

2) pascal.kln
klein program to print the nth row of pascals triangle
